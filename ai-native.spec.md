# AI-Native Specification

## 1. Definition
AI-Native means the learning experience is designed
around AI thinking patterns rather than traditional lectures.

---

## 2. AI Agent Concepts
- Perception Agent
- Decision Agent
- Planning Agent
- Safety Agent

Each agent is taught as a conceptual module.

---

## 3. Prompt-Based Learning
- Learners are given structured prompts
- Prompts simulate AI reasoning
- No execution required

---

## 4. Simulation Thinking
- Mental simulations over physical simulations
- Flow diagrams described textually
- Environment-agent interaction models

---

## 5. AI Without APIs
- No runtime inference
- No API calls
- No dependency failures

---

## 6. AI-Native Exercises
- "Think like an AI Agent"
- "Design a decision loop"
- "Predict robot behavior"

---
