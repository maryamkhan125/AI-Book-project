# Course Specification: Physical AI & Humanoid Robotics

## 1. Course Level
Beginner to Intermediate

---

## 2. Duration
16 Weeks (One Semester)

---

## 3. Weekly Breakdown

- Weeks 1–2: Introduction to Physical AI
- Weeks 3–4: Embodied Intelligence
- Weeks 5–6: Sensors and Actuators
- Weeks 7–8: Humanoid Robot Design
- Weeks 9–10: AI Agents in Robotics
- Weeks 11–12: Planning and Control
- Weeks 13–14: Human-Robot Interaction
- Weeks 15–16: Ethics and Future Applications

---

## 4. Teaching Methodology
- Conceptual lectures
- Thought experiments
- Prompt-driven exercises
- System design discussions

---

## 5. Assessments
- Conceptual quizzes
- Design assignments
- Final system proposal (no hardware required)

---

## 6. Tools
- Docusaurus
- Markdown / MDX
- Diagrams as code blocks
